Run 'enable default administrator account.bat' as Admin
Sign out and log in as Administrator
Rename User Folder in 'C:\Users'
Regedit to 'HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileList\' and find the SID for your User
Edit the ProfileImagePath key to reflect the new folder name
