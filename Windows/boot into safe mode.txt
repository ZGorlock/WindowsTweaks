Run 'boot into safe mode.bat' as Admin

Alternatively:
	Run 'msconfig.exe'
		Switch to the 'Boot' tab
		Under Boot options:
			Enable 'Safe boot'
		Click 'Ok'
	Restart computer

Alternatively:
	Open the default Windows 10 start menu
		Hold Shift
		Click Power
			Click Restart
	On Boot Screen:
		Select 'Troubleshoot'
		Select 'Advanced options'
		Select 'Startup Settings'
		Click Restart
	On Startup Screen:
		Select the entry for 'Enable Safe Mode'
