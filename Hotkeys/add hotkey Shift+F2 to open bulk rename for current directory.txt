Read 'enable hotkeys.txt'

This assumes you have Bulk Rename Utility installed at: 'C:\Program Files\Bulk Rename Utility'
	You can download this program from: https://www.bulkrenameutility.co.uk/

Navigate to 'ahk'
Run 'BulkRenameHere.ahk'

This will add the following hotkeys:
	Shift+F2			Open BulkRenameUtility for open explorer directory

To run at startup:
	Create a shortcut to 'BulkRenameHere.ahk' and copy to: 'C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp'
