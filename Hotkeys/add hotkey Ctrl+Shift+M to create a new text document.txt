Read 'enable hotkeys.txt'

Navigate to 'ahk'
Run 'NewTextDocument.ahk'

To run at startup:
	Create a shortcut to 'NewTextDocument.ahk' and copy to: 'C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp'
