Navigate to 'LibreOffice'

To import the color palettes:
	Copy the files from 'LibreOffice\palette' to 'C:\Program Files\LibreOffice\share\palette'
	Run 'LibreOffice'
		Click: Background Color
		Select: favorite

To import the color scheme:
	Edit 'C:\Users\<User>\AppData\Roaming\LibreOffice\4\user\registrymodifications.xcu'
		Find the "ColorScheme/ColorSchemes" section in registrymodifications.xcu
		Copy and paste the content from 'LibreOffice\scheme\dracula.xcu'
	Run 'LibreOffice'
		Open: Tools / Options
		Under: LibreOffice / Application Colors
			Select: Dracula
			Click Ok
