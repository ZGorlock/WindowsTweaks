Navigate to 'IntelliJ'

Copy 'Dark Custom.icls' and 'Darcula.icls' to: 'C:\Users\<User>\AppData\Roaming\JetBrains\IdeaIC2023.3\colors\'

Copy 'filetypes\ahk.xml' to: 'C:\Users\<User>\AppData\Roaming\JetBrains\IdeaIC2023.3\filetypes\'

Launch: IntelliJ
	Open: File / Settings
	Under: Editor / Color Scheme / General
		For 'Scheme' Select: 'Dark Custom'
		Click Ok
	Under: Editor / File Types
		Select 'Recognized File Type': 'Autohotkey' or 'ahk'
		Add 'File name pattern': *.ahk
		Click Ok
